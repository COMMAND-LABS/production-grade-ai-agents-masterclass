## Prerequisites

- A laptop or Desktop
- Docker
- VSCode or Cursor
- 2 "VSCode" extensions
  - the "Docker" extension and the "Dev containers" extension
- a GitHub account
- a GCP account

## How to run this project

- Open the project with VSCode
- SHIFT + COMMAND + P
  - Reopen in Container
- populate `.env.template` with required environment variables
- `pip install -r requirements.txt`
- `python main.py`
